C++ Writing Style

Correct

#include <iostream> #include <string> using namespace std; int main() { string message = "Hello World!\n"; cout << message; return 0; }

Wrong

using namespace std; #include <iostream> string M = "Hello"; int main() { cout << M;; return 3; }

Naming Conventions

  • Names cannot start with digits
  • Avoid C++ keywords
  • PascalCase → classes, functions
  • snake_case → variables

Variables & Type Conversion

double a = 3.5; int b = (int) a; // b = 3
